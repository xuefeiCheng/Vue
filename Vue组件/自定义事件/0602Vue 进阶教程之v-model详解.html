首先一点 看到这个v-model语法糖，就会很自燃的想到angularjs中的ng-model指令。但是他们是有本质区别的， v-model只是vue.js中的一个语法糖，并不是ng-model用于数据双向绑定的指令。

在 Vue 中，有许多方法和 Angular 相似，这主要是因为 Angular 是 Vue 早期开发的灵感来源。然而，Augular 中存在许多问题，在 Vue 中已经得到解决。
